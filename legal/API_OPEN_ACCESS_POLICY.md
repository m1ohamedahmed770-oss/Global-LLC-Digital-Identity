# سياسة الوصول المفتوح للواجهات البرمجية (API Open Access Policy) لشركة Global LLC

## 1. مبدأ الوصول المفتوح

تلتزم Global LLC بتوفير واجهات برمجية تطبيقية (APIs) مفتوحة الوصول للمجتمع الرقمي، بهدف تعزيز الابتكار والمساهمة في تطوير المشاريع التقنية.

## 2. شروط الاستخدام

*   **الاستخدام غير التجاري:** تُخصص واجهات API بشكل أساسي للاستخدام **غير التجاري** والمشاريع المفتوحة المصدر.
*   **التوقيع الإلزامي:** يجب على أي مشروع يستخدم أو يعدل على واجهات API الخاصة بـ Global LLC أن يلتزم بمتطلبات التوقيع الرقمي والإسناد الموضحة في سياسة الملكية الرقمية.
*   **الحدود:** تحتفظ Global LLC بالحق في فرض حدود على معدل الاستخدام (Rate Limiting) لضمان استقرار الخدمة لجميع المستخدمين.

## 3. قواعد التعديل والإسناد

*   **الإسناد عند التعديل:** عند تعديل أي واجهة API، يجب إضافة العبارة التالية في التوثيق أو رأس الاستجابة (Response Header):
    `"ProjectName — modified by GlobalLLC.community"`
*   **الترخيص:** يتم ترخيص واجهات API تحت تراخيص مفتوحة المصدر (يتم تحديدها لكل API على حدة)، ويجب الالتزام بشروط هذا الترخيص.
